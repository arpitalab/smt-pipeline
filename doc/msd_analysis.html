MSD Analysis

getMSD computes the ensemble time-averaged MSD across all tracks, then fits a fractional Brownian motion model to the result using bootstrap resampling to quantify uncertainty.

getMSD / recomputeMSD

results = tc.getMSD('TrackType','relative', 'MaxLag',40, 'ExposureTime',0.01, ... 'MinLength',40, 'MinMeanSqStep',4*sigma_fixed^2, ... 'NumBootstrap',100, 'Condition','Control') results = tc.recomputeMSD(...) % same args; forces recomputation

How MSD is computed

For each track, gaps are filled with NaN, then simple_msd() computes the time-averaged MSD over all overlapping pairs. The ensemble mean is a weighted average (weighted by number of valid observations at each lag).

How the fBM fit works

Each bootstrap replicate fits the model: 

MSD(n·dt) = 4·G·ψ(n, frac, α) + 4·σ²

where ψ is the Backlund exposure-time kernel and the fit uses lsqnonlin over the first floor(MaxLag/2) lags. Parameters: G (generalized diffusion, µm²/s^α), σ² (localization noise, µm²), α (anomalous exponent). Bounds: G ∈ [1e-5, 5], σ² ∈ [1e-6, 4e-3], α ∈ [0.1, 1.5].

MSD fit vs Toeplitz MLE: The MSD least-squares fit uses only the marginal variance at each lag (the diagonal of the covariance matrix). The Toeplitz MLE (getFBMParameters) uses the full covariance structure including anti-correlations between consecutive steps. The MLE estimate of α is generally more accurate for short tracks. Use plotFBMFit to compare both on the same axes.

plotMSD

figs = tc.plotMSD() figs = tc.plotMSD('Color',[0 0.4 0.8], 'Title','H2B Control', 'ShowBootstraps',false)

Requires getMSD() to have been run. Produces a log-log figure with:

  • Shaded band: 2.5–97.5th percentile of bootstrap means (95% CI on the mean MSD).
  • Solid line: empirical ensemble mean.
  • Dashed line: median bootstrap fBM fit curve, with α and its 95% CI in the legend.
  • Optional: individual bootstrap fit curves when 'ShowBootstraps',true.

Parameters

ParameterDefaultDescription
Color[0 0.4 0.8]RGB color for all MSD elements.
TitleautoFigure title. If empty: "Ensemble MSD (n=N tracks)".
ShowBootstrapsfalseOverlay individual bootstrap fit curves (semi-transparent).

Typical workflow

sigma_fixed = 0.040;   % µm — measured from fixed beads

msd_opts = struct( ...
    'TrackType',     'relative',      ...
    'MinLength',      40,             ...
    'MinMeanSqStep',  4*sigma_fixed^2,...
    'MaxLag',         40,            ...
    'NumBootstrap',   100,           ...
    'ExposureTime',   0.01);

msd = tc.recomputeMSD(msd_opts);

tc.plotMSD('Title','H2B Control');

fprintf('Median alpha = %.3f\n', median(msd.boot_fits(~any(isnan(msd.boot_fits),2), 3)));

Matching MSD and fBM populations

If you plan to overlay the Toeplitz MLE curve via plotFBMFit, set MinLength in getMSD to match SubtrackLength in getFBMParameters. Short tracks contribute to the MSD but are excluded from the MLE (they don't form complete subtracks), producing a vertical offset between the two curves. See Filtering for details.
