SMTDatabase

SMTDatabase is a SQLite-backed registry for SMT experiments. It tracks which TrajectoryCollection objects exist, which analyses have been run, and key numerical summaries — without duplicating the full data (which stays in .mat files on disk).

Requires MATLAB R2022b+ (uses the built-in sqlite function; no Database Toolbox needed).

Data hierarchy

Collection  – one per molecule / treatment / substrate / imaging rate.
              Spans ALL dates and FOVs for that condition group.
              Example: "H2B_E2_1kPa_slow"

Experiment  – one per FOV (TrajectoryWrapper).
              Date auto-extracted from the file path.
              Many experiments → one collection.

Analysis run – one per completed analysis (pEM, MSD, Lifetime, fBM).
               Stores key numerical summaries as JSON; full results
               live in the .mat pointed to by the collection.

Constructor

db = SMTDatabase('smt_pipeline.db')   % opens or creates the .db file

Close the connection explicitly when done:

db.close()   % or just clear db (destructor closes it)

registerCollection

Registers a TrajectoryCollection and all its FOVs in the database. Physical metadata (molecule, substrate, imaging rate, exposure time) is auto-parsed from file paths using the HILO directory convention: 

HILO/{molecule}/{e-i}/{substrate}/{YYYYMMDD}/quot_input_links/*.csv
collId = db.registerCollection(tc, 'Name','H2B_E2_1kPa_slow', 'Treatment','E2') % Preview path parsing before registering: SMTDatabase.parsePathMetadata('/path/to/file_trajs.csv')

Parameters

ParameterRequired?Description
NameYesUnique collection name, e.g. 'H2B_E2_1kPa_slow'.
TreatmentYesTreatment label (cannot be inferred from path).
MoleculeautoAuto-parsed from path; override if needed.
SubstrateautoAuto-parsed from path.
ImagingHzautoAuto-parsed from filename (i200ms → 5 Hz).
ExposureTimeautoAuto-parsed from filename (e10ms → 0.01 s).
MatPathNoPath to the .mat file with full results (set after saving).
TomlFileNoPath to experiment TOML file.
NotesNoFree-text notes.
Duplicate names: Registration fails if the collection name already exists. Use updateCollection to change metadata on an existing entry.

updateCollection

db.updateCollection('H2B_E2_1kPa_slow', ...
    'MatPath',      'results/H2B_E2_1kPa_slow.mat', ...
    'ExposureTime', 0.01, ...
    'Notes',        're-analyzed with SubtrackLength=40')

Updatable fields: MatPath, Notes, Molecule, Treatment, Substrate, ImagingHz, ExposureTime, TomlFile.

logAnalysis

Records a completed analysis run with key numerical summaries.

db.logAnalysis('H2B_E2_1kPa_slow', tc, 'pEM')
db.logAnalysis('H2B_E2_1kPa_slow', tc, 'MSD')
db.logAnalysis('H2B_E2_1kPa_slow', tc, 'Lifetime')
db.logAnalysis('H2B_E2_1kPa_slow', tc, 'pEM', ...
    'ConditionFilter', 'E2_treated', ...
    'Notes',           'K=3, bootstrap 500 resamples')

The mat_path stored on the collection is automatically referenced. Summaries (optimal K, state D values, proportions) are stored as JSON in the analysis_runs table.

Browsing and querying

% List all collections
db.listCollections()

% Filter by fields
db.listCollections('Molecule','H2B', 'Substrate','1kPa')

% List analysis runs
db.listAnalyses('AnalysisType','pEM')

% Get full info for one collection
info = db.getCollection('H2B_E2_1kPa_slow');
% info.collection  – metadata row
% info.experiments – FOV table with exp_date per FOV
% info.analyses    – analysis run history
% info.pEM         – decoded state summaries (if pEM was logged)

% Load a TrajectoryCollection from its registered .mat
tc = db.loadCollection('H2B_E2_1kPa_slow');

% Raw SQL query
T = db.query('SELECT * FROM analysis_runs WHERE optimal_k = 3')

Typical end-to-end workflow

% 1. Create database
db = SMTDatabase('smt_pipeline.db');

% 2. Load and analyse
tc = TrajectoryCollection();
tc.ReadParams('experiment.toml');
files = dir('data/HILO/H2B/e10ms_i200ms/1kPa/20240315/**/*.csv');
for k = 1:numel(files)
    tc.addFromFile(fullfile(files(k).folder, files(k).name), ...
        'FileID', num2str(k), 'Condition', 'E2');
end
tc.getBayesianDiffusivity();
tc.computeBootstrapCI('nBoot', 200);
tc.getFBMParameters('TrackType','relative','ExposureFraction',0.05, ...
    'SubtrackLength',40,'CIMethod','profile');

% 3. Register and log
db.registerCollection(tc, 'Name','H2B_E2_1kPa_slow', 'Treatment','E2');

save('results/H2B_E2_1kPa_slow.mat', 'tc');
db.updateCollection('H2B_E2_1kPa_slow', 'MatPath','results/H2B_E2_1kPa_slow.mat');
db.logAnalysis('H2B_E2_1kPa_slow', tc, 'pEM');

% 4. Browse later
db.listCollections();
info = db.getCollection('H2B_E2_1kPa_slow');
disp(info.pEM);   % state fractions and D values
