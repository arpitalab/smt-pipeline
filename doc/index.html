smt-pipeline documentation

MATLAB toolbox for single-molecule tracking (SMT) analysis built around TrajectoryCollection. Trajectories are loaded from CSV files (quot) or SMD objects and flow into MSD, Richardson-Lucy, Bayesian diffusivity, and fBM maximum-likelihood analyses.

New in this release

fBM Analysis

Toeplitz MLE parameter estimation, CI computation, and plotFBMFit — overlay the MLE model curve on the empirical ensemble MSD.

Track Filtering

New MinLength and MinMeanSqStep options in getMSD and getFBMParameters to exclude short tracks and immobile particles.

Struct Options

Pass a MATLAB struct instead of long name-value lists to any analysis method. Define options once, reuse across conditions.

Core methods at a glance

MethodWhat it does
getMSD / recomputeMSD Ensemble time-averaged MSD with bootstrap confidence intervals and least-squares fBM curve fit.
getFBMParameters Population Toeplitz MLE of K, α, σ with profile-likelihood or MCMC confidence intervals.
plotMSD Log-log plot of ensemble MSD with bootstrap CI shading and median fit curve.
plotFBMFit Overlay the Toeplitz MLE model curve on the empirical MSD for direct comparison of the two estimation methods.
getRLDecomposition Richardson-Lucy MSD distribution decomposition via RL_HILO.
getFBMByRLState Per-RL-state Toeplitz MLE fBM fits.
getFBMAlphaDistribution Per-track (K, α) distribution with optional fixed σ.
getBayesianDiffusivity Bayesian (pEMv2) diffusivity classification.
getLifetime Fluorophore lifetime distribution with exponential fits.
