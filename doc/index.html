smt-pipeline documentation

MATLAB toolbox for single-molecule tracking (SMT) analysis built around TrajectoryCollection and TrajectoryWrapper. Trajectories are loaded from CSV files or SMD objects and flow into MSD, Richardson-Lucy, Bayesian diffusivity, fBM maximum-likelihood, and lifetime analyses.

Pipeline overview

                 CSV files                   SMD objects
                    │                             │
                    ▼                             ▼
            TrajectoryWrapper ◄──────── TrajectoryWrapper.fromSMD()
          (load → cull → relative)
                    │
                    ▼
          TrajectoryCollection       ← aggregate N wrappers / conditions
                    │
          ┌─────────┼──────────────────────────────────┐

        getMSD  getRLDecomp  getBayesian  getFBMParam  getLifetime
        plotMSD  getFBMByRL   Diffusivity  getFBMAlpha   computeDACF
               plotRLDecomp  computeBootCI plotFBMFit
                              plotpEMstats

Quick start

run('setup_path.m');

% --- Load data -----------------------------------------------------------
tc = TrajectoryCollection();
tc.ReadParams('experiment.toml');           % sets PixelSize, FrameInterval
files = dir('data/*.csv');
for k = 1:numel(files)
    tc.addFromFile(fullfile(files(k).folder, files(k).name), ...
                   'FileID', num2str(k), 'Condition', 'Control');
end
tc.summary();

% --- MSD -----------------------------------------------------------------
msd = tc.recomputeMSD('TrackType','relative','MaxLag',40,'ExposureTime',0.01, ...
                       'MinLength',40,'MinMeanSqStep',4*0.040^2,'NumBootstrap',100);
tc.plotMSD('Title','H2B Control');

% --- fBM MLE -------------------------------------------------------------
fbm = tc.getFBMParameters('TrackType','relative','ExposureFraction',0.05, ...
                            'SubtrackLength',40,'MinSubtrackLength',40, ...
                            'MinMeanSqStep',4*0.040^2,'CIMethod','profile+mcmc');
tc.plotFBMFit('FBMResults',fbm,'Title','H2B Control');

% --- RL decomposition ----------------------------------------------------
rl = tc.getRLDecomposition('TrackType','relative','LagTime',4,'String','H2B');
tc.plotRLDecomposition();

% --- Bayesian diffusivity (pEMv2) ----------------------------------------
tc.getBayesianDiffusivity('Condition','Control');
tc.computeBootstrapCI('nBoot',200);

Core classes

ClassRole
TrajectoryWrapper Single FOV: loads CSV or SMD, culls tracks (ROI + gap filter), removes global cell motion via maximum spanning forest.
TrajectoryCollection Aggregates N wrappers across conditions. All analysis methods live here.
SMTDatabase SQLite registry: registers collections, logs analysis runs, enables cross-experiment queries.

All analysis methods

MethodPageWhat it does
getMSD / recomputeMSD MSD Ensemble time-averaged MSD with bootstrap CI and least-squares fBM curve fit.
plotMSD MSD Log-log MSD plot with CI shading and median fit curve.
getFBMParameters fBM Population Toeplitz MLE of K, α, σ with profile-likelihood or MCMC confidence intervals.
getFBMAlphaDistribution fBM Per-track (K, α) distribution; sigma optionally fixed to population value.
plotFBMFit fBM Overlay Toeplitz MLE model curve on empirical ensemble MSD.
getRLDecomposition RL Richardson-Lucy MSD distribution decomposition via RL_HILO.
getFBMByRLState RL Population fBM MLE run independently on each RL-classified state.
plotRLDecomposition RL Reproduce van Hove, P(MSD), and per-state MSD figures.
getBayesianDiffusivity pEM Bayesian (pEMv2) diffusivity classification into optimal number of states.
computeBootstrapCI pEM Bootstrap confidence intervals on pEM state fractions and diffusivities.
plotpEMstats pEM Per-state MSD plots from pEM classification.
getLifetime Lifetime Bound-particle dwell-time distribution with exponential fits.
computeDACF Lifetime & DACF Displacement autocorrelation function — diagnostic for fBM anomalous exponent.

Key parameters and track types

TrackTypeDescription
'culled'Tracks that passed ROI membership and gap-consistency culling (default for all methods).
'relative'Culled tracks with global cell motion subtracted via maximum spanning forest. Use for fBM analysis to remove correlated drift.
'raw'All loaded tracks before any filtering.
