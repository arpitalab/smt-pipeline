Lifetime & DACF

getLifetime

Estimates the dwell-time (fluorophore lifetime) distribution by detecting bound particles — molecules that remain confined to a small region — via the find_bound_particles_2 algorithm.

results = tc.getLifetime() results = tc.getLifetime('rmin',0.25, 'rmax',0.4, 'minTrackLength',4, 'maxTrackLength',400)

recomputeLifetime forces recomputation with new parameters: 

tc.recomputeLifetime('rmin', 0.20, 'rmax', 0.35);

Parameters

ParameterDefaultDescription
rmin0.25Inner confinement radius (µm). Molecules within this radius across consecutive frames are considered bound.
rmax0.4Outer confinement radius (µm). Tracks that exit beyond rmax are not classified as bound.
minTrackLength4Minimum track length in frames to include in the lifetime analysis.
maxTrackLength400Maximum track length in frames (excludes very long bleaching artefacts).
UseRawfalseUse raw (unculled) tracks instead of culled tracks.

Result fields

FieldDescription
.xTrack length axis (frames).
.fEmpirical lifetime survival function (proportion of bound events lasting ≥ x frames).
.floLower 95% confidence bound on f.
.fupUpper 95% confidence bound on f.
.TrackLengthsRaw track lengths for all tracks that entered the analysis.

Example

tc.getLifetime('rmin', 0.25, 'rmax', 0.4);

x   = tc.LifetimeResults.x;
f   = tc.LifetimeResults.f;
flo = tc.LifetimeResults.flo;
fup = tc.LifetimeResults.fup;
dt  = tc.getFrameIntervalForCondition([]);

figure;
fill([x; flipud(x)] * dt, [fup; flipud(flo)], [0.7 0.9 1], ...
     'EdgeColor','none', 'FaceAlpha', 0.5);
hold on;
semilogy(x * dt, f, 'b-', 'LineWidth', 2);
set(gca, 'YScale','log', 'FontSize',16);
xlabel('Dwell time (s)');
ylabel('Survival probability');

computeDACF — Displacement Autocorrelation

The displacement autocorrelation function (DACF) is a model-free diagnostic for anomalous diffusion. For a pure fBM process with anomalous exponent α: 

DACF(k=1) = (2^α − 2) / 2

Values near −0.5 indicate strong anti-correlation (subdiffusion, α ≈ 0.5). Values near 0 indicate uncorrelated steps (Brownian, α = 1). Positive values indicate super-diffusion. The DACF is particularly useful for validating fBM parameter estimates from getFBMParameters.

[dacf, lags] = tc.computeDACF() [dacf, lags] = tc.computeDACF('MaxLag',5, 'MinStepVar',4e-4, 'TrackType','relative')

Parameters

ParameterDefaultDescription
TrackType'culled''culled' | 'relative' | 'raw'
Condition[]Restrict to a condition.
MaxLag5Maximum lag in frames for the ACF.
MinTrackLength10Minimum track length (frames) to include.
MinStepVar0Minimum mean squared step (µm²). Near-immobile tracks are excluded as their DACF is undefined (division by near-zero variance). Set to ~4·σ².

Output

OutputSizeDescription
dacfMaxLag × 1Empirical DACF at lags 1 … MaxLag.
lagsMaxLag × 1Lag indices.

Example — comparing empirical DACF to fBM prediction

sigma_fixed = 0.040;   % µm
[dacf, lags] = tc.computeDACF( ...
    'TrackType',    'relative', ...
    'MaxLag',        5,         ...
    'MinStepVar',    4*sigma_fixed^2);

% Get alpha from population fBM MLE
fbm = tc.getFBMParameters('TrackType','relative', ...
    'ExposureFraction',0.05,'SubtrackLength',40,'CIMethod','none');

dacf_theory = (2.^(lags * fbm.alpha) - 2) / 2;   % approximate fBM prediction

figure;
plot(lags, dacf, 'ko-', 'LineWidth',2, 'MarkerFaceColor','k', 'DisplayName','Empirical');
hold on;
plot(lags, dacf_theory, 'r--', 'LineWidth',1.5, 'DisplayName',sprintf('fBM (α=%.2f)',fbm.alpha));
xlabel('Lag (frames)');
ylabel('DACF');
legend('Box','off');
title('Displacement autocorrelation');
Interpreting DACF: If the empirical DACF(1) matches (2^α − 2)/2 from getFBMParameters, the fBM model is consistent with the data. A discrepancy suggests either residual immobile particles (pull DACF toward 0) or residual correlated drift in relative tracks (pull DACF positive).
