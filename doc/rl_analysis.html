Richardson-Lucy (RL) Analysis

The RL analysis deconvolves the distribution of single-molecule MSDs to reveal the underlying population of diffusive states. The pipeline wraps RL_HILO, which computes the van Hove correlation function and applies Richardson-Lucy deconvolution to recover the MSD probability density P(MSD). Tracks are then classified into states and optionally fit with per-state fBM MLEs.

getRLDecomposition

results = tc.getRLDecomposition('TrackType','relative', 'LagTime',4, 'String','H2B') results = tc.getRLDecomposition(opts_struct)

Parameters

ParameterDefaultDescription
TrackType'culled''culled' | 'relative' | 'raw'
LagTime4Frame lag used to compute the van Hove correlation G(r, τ). Typical values: 4–8.
String'Not specified'Label used in figure titles produced by RL_HILO.
Condition[]Restrict to a condition string.
ExposureFraction1Te/dt passed to RL_HILO's internal fBM model. Default 1 (stroboscopic).
MinStepVar0Pre-filter: exclude tracks with mean squared step < this value (µm²). Use ~4·σ² to drop immobile particles before RL runs.
MinTrackLength7Minimum track length in frames (tracks shorter than this are dropped).
MinGroupSize50Minimum tracks per state for the internal RL fBM fit to be attempted.
SubtrackLength20Subtrack length for the per-state fBM MLE run inside RL_HILO.
CIMethod'none'CI method for the internal RL fBM fits. For full CIs use getFBMByRLState after this call.
ForceRecomputefalseRerun even if cached. Required when switching TrackType.

What RL_HILO does

  1. Computes the van Hove displacement distribution G(r, τ) at lag LagTime.
  2. Deconvolves with Richardson-Lucy to recover P(MSD) — the distribution of single-molecule MSDs.
  3. Identifies peaks in P(MSD) corresponding to discrete diffusive states.
  4. Classifies each track into one state based on its MSD at the selected lag.
  5. Runs a quick per-state fBM MLE (with CIMethod='none' by default).

Result fields

FieldDescription
.MMSD axis of the P(MSD) distribution.
.computed_quantitiesStruct from RL_HILO: contains .classified_tracks (cell array, one per state), .msd, .msderr, .vanHove, .Gs, .P1norm, .x, .ident.
.dtFrame interval (s).
.fracExposureFraction used.
.track_typeWhich track set was used.
.track_index_mapMaps filtered-array indices → full track array indices (needed by getFBMByRLState).
Caching: Results are cached on the object. Re-calling with the same TrackType returns the cached result. Switching TrackType without ForceRecompute=true produces a warning.

Example

sigma_fixed = 0.040;   % µm

rl = tc.getRLDecomposition( ...
    'TrackType',      'relative', ...
    'LagTime',         4,         ...
    'String',         'H2B',      ...
    'MinStepVar',      4*sigma_fixed^2, ...
    'MinTrackLength',  20,        ...
    'ExposureFraction', 0.05,     ...
    'SubtrackLength',  20);

fprintf('Number of RL states: %d\n', numel(rl.computed_quantities.classified_tracks));
for s = 1:numel(rl.computed_quantities.classified_tracks)
    n = numel(rl.computed_quantities.classified_tracks{s});
    fprintf('  State %d: %d tracks\n', s, n);
end

getFBMByRLState

Runs the population Toeplitz MLE (fitFBM_MLE) independently on each RL-classified state. The internal RL fBM fits use CIMethod='none'; call this method for full profile or MCMC confidence intervals. Must be called after getRLDecomposition().

results = tc.getFBMByRLState() results = tc.getFBMByRLState('CIMethod','profile', 'MinGroupSize',50, 'PerTrack',true)

Parameters

ParameterDefaultDescription
CIMethod'profile''profile' | 'profile+mcmc' | 'mcmc' | 'hessian' | 'none'
SubtrackLength20Subtrack length for population MLE per state.
MinGroupSize50States with fewer tracks are skipped.
MinStepVar0Pre-filter within each state (µm²).
MinAlpha0Post-fit alpha floor for per-track results (when PerTrack=true).
PerTrackfalseAlso run fitFBM_pertracks per state for per-track α distributions.
MaxSubtrackLength20Subtrack cap for per-track fits (same as SubtrackLength by default).
VerbosefalsePrint per-state fitting progress.

Returns

A struct array results(s) with one entry per state:

FieldDescription
.stateState index.
.n_tracksNumber of tracks in this state.
.K, .alpha, .Ka, .sigmaPopulation MLE estimates.
.CIConfidence interval struct (from fitFBM_MLE).
.pertracksPer-track result struct (if PerTrack=true).
.skippedtrue if state was too small or fitting failed.

Example

% Run RL first (see above)
fbm_states = tc.getFBMByRLState( ...
    'CIMethod',      'profile', ...
    'SubtrackLength', 40,       ...
    'MinGroupSize',   50,       ...
    'PerTrack',       true);

for s = 1:numel(fbm_states)
    if ~fbm_states(s).skipped
        fprintf('State %d: alpha=%.3f [%.3f, %.3f]  K=%.4g\n', ...
            s, fbm_states(s).alpha, ...
            fbm_states(s).CI.alpha(1), fbm_states(s).CI.alpha(2), ...
            fbm_states(s).K);
    end
end

plotRLDecomposition

Reproduces the standard three-panel RL analysis output from stored results without re-running RL_HILO.

figs = tc.plotRLDecomposition() figs = tc.plotRLDecomposition('Title','H2B Control')

Produces three figures:

FigureContents
Fig 1van Hove correlation G(r, τ): data (circles) + RL fit (line). Log-linear scale.
Fig 2P(MSD): Richardson-Lucy deconvolved MSD distribution. Semi-log scale.
Fig 3Per-state ensemble MSD with 99% CI error bars, log-log scale. Each state colored separately.
