Bayesian Diffusivity (pEM)

The pEM analysis uses pEMv2_SPT — a permutation expectation-maximisation algorithm — to classify tracks into a data-driven optimal number of diffusive states and estimate each state's diffusion coefficient and localization noise. No prior assumption about the number of states is needed.

getBayesianDiffusivity

results = tc.getBayesianDiffusivity() results = tc.getBayesianDiffusivity('Condition','Control', 'SplitLength',7)

Parameters

ParameterDefaultDescription
Condition[]Restrict to one condition. When set, results are not cached on the object.
MinLength6Minimum track length in frames before splitting.
SplitLength7Track split length (frames). Tracks are split into non-overlapping segments before pEM runs. Shorter splits increase the number of subtracks but reduce the number of features available per segment.
NumFeaturesSplitLength−1Number of velocity autocorrelation features used. Capped at 10. Default is SplitLength−1.
ForceRecompute[]Force rerun (auto-true on first call or when Condition is set).
SplitLength guidance: The default SplitLength=7 is appropriate for pEM but is too short for fBM analysis (alpha collapses toward 1 with short subtracks). For getFBMByRLState or getFBMParameters use SubtrackLength ≥ 20.

Algorithm internals

pEMv2 builds a velocity autocorrelation (VACF) feature vector for each split subtrack, then runs EM over K=1…8 Gaussian mixture states, initializing numReinitialize=20 times and perturbing numPerturbation=200 times to escape local optima. BIC selects the optimal K. The algorithm reports:

  • optimalD — diffusion coefficient per state (µm²/s).
  • optimalS — localization noise σ per state (µm).
  • optimalP — population fraction per state.
  • posteriorProb — N_subtracks × K posterior probability matrix.

Result structure

Stored in tc.pEMResults. Key fields:

FieldDescription
.pEMTableMATLAB table; one row. Contains optimalD, optimalS, optimalP, optimalVacf, posteriorProb, optimalSize (selected K), BIC, splitX, trackID, splitID.
.proportionsTable: State, Proportion, OptimalD, OptimalS — one row per state, sorted by D.

Accessing results

tc.getBayesianDiffusivity('Condition','Control');

% Population fractions and diffusivities
prop = tc.pEMResults.proportions;
disp(prop);

% Per-subtrack classification
pp        = tc.pEMResults.pEMTable.posteriorProb{1};
[~, state] = max(pp, [], 2);   % most probable state for each subtrack

% Posterior confidence filter (optional)
max_pp     = max(pp, [], 2);
conf       = max_pp > 0.5;     % high-confidence assignments only

% Diffusion coefficients (sorted ascending)
D = tc.pEMResults.pEMTable.optimalD{1};   % µm²/s

computeBootstrapCI

Computes confidence intervals on pEM state fractions and diffusion coefficients by resampling parent tracks and re-running pEMv2. Must be called after getBayesianDiffusivity() on the full collection (without a Condition filter).

ciResults = tc.computeBootstrapCI() ciResults = tc.computeBootstrapCI('nBoot',200, 'parallel',true, 'resampleLevel','parent')

Parameters

ParameterDefaultDescription
nBoot100Number of bootstrap resamples.
covModel'normal'Covariance model: 'normal' | 'confined' | 'fBM'.
alpha0.05CI level: 1−alpha (e.g. 0.05 → 95% CI).
nRandomStarts5EM restarts per bootstrap replicate.
resampleLevel'parent'Resample at parent track level ('parent') or subtrack level ('subtracks').
paralleltrueUse parfor (requires Parallel Computing Toolbox).
verbosetruePrint bootstrap progress.

Results are stored in tc.pEMResults.ciresults.

tc.getBayesianDiffusivity();         % must run on full collection first
ci = tc.computeBootstrapCI('nBoot', 200, 'parallel', true);
% ci contains CI bounds on D, S, and population fractions per state

plotpEMstats

fig = tc.plotpEMstats()

Produces a per-state ensemble MSD plot using the pEM-classified subtracks. States with population fraction < 2.5% or posterior probability < 0.5 are excluded. State MSD curves are plotted as mean ± SEM error bars.

Hardcoded split length: plotpEMstats currently uses a hardcoded splitLength = 7 (matching the pEM default). The x-axis shows lags from 0 to 6 frames.

Typical workflow

% Run pEM on all culled tracks
tc.getBayesianDiffusivity();

% Check proportions
disp(tc.pEMResults.proportions);

% Run bootstrap CI (parallel if pool available)
tc.computeBootstrapCI('nBoot', 200);

% Plot per-state MSD
tc.plotpEMstats();

% Save full results
save('results/H2B_control.mat', '-struct', 'tc');  % or use SMTDatabase
